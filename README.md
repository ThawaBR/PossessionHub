# Possession Hub

Track possessed players and access upcoming features in one hub.

## How to use

Copy and paste this code into your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/ThawaBR/PossessionHub/refs/heads/main/source"))()
```
## Features
- Player:
  - Speed.
  - Respawn.
  - Noclip.
  - Anti Mute.
  - Chat Log.
  - Redeem Codes.
  - Items Esp.
  - Possessor Esp.
  - Safe Zone.
  - Auto Collect.
- Emotes: 56+ animations available: Levitate, Thriller, RhythmRiot, and many more.
- Animations: 14+ animations available: Standing, Slouched, Shy, and many more.
- Roulette: Allows you to pick any item from the roulette.
- Shop:
  - Auto Buy Spin.
  - Auto Spin.
  - Jade Contemplation.
  - Jade Shatter.
- Troll:
  - Pizza Sound Spam.
  - Lag.
  - Glow.
  - Follow Player.
- Settings:
  - Anti Kick.
  - Anti Afk.
  - Rejoin.
  - ServerHop.
